# Django Login System

This project is a user authentication and profile management system built with Django, featuring user signup, login, and profile updates. It utilizes REST APIs for seamless user data management and supports full CRUD operations.


### Features:
- User Registration: Allows users to register with a unique username, email, and password.
- User Login: Secure login functionality with token-based authentication.
- Profile Management: Users can update or delete their profiles.
- CRUD Operations: Full Create, Read, Update, and Delete operations for managing user data using id as the primary key.
- REST API Integration: Exposes RESTful API endpoints for user management.

### Technologies:
- Django Framework: Backend logic, database management, and REST API implementation.
- Django REST Framework (DRF): To build and manage RESTful APIs.
- Postman: Used for testing and validating API endpoints.
